# Ecommerce Purchase Analysis (Pandas)

This project analyzes ecommerce purchase data using Python and pandas.

The goal is to explore basic business questions such as:
- How much revenue was generated?
- Which products and categories performed best?
- Which customers spent the most?

## Tools Used
- Python
- pandas

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python ecommerce_analysis.py

## Notes
This project is intentionally kept simple and focused on practical data analysis rather than advanced modeling.
